About Me Dashboard 🌟

An interactive Plotly Dash web app showcasing my professional journey and skills.

This dashboard transforms a traditional resume into a modern, user-friendly experience. Featuring sections for “Professional Experience,” “About Me,” and a dynamic gallery, it also includes live weather updates and a dark mode toggle for a personalized touch. Built with a focus on clean design and functionality, this app is perfect for sharing with colleagues and hiring managers.
